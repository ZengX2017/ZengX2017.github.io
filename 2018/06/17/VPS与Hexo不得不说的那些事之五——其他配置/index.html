PM2使用相关

为了让Hexo能保持后台运行，使用PM2对其进行管理

安装pm2

npm  install -g pm2

写一个执行脚本

在博客根目录下面创建一个xxx(此处命名为：hexo_run).js

在根目录下运行脚本

pm2 start hexo_run.js
